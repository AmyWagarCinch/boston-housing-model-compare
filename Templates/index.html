Home Value Prediction Tool

Using the Boston Dataset to evaluate testing accuracy of machine learning models incorporating a range of factors.

This is a Machine Learning project done to compare different models using basic Linear Regression and Random forest Machine Learning concepts. The Boston Dataset was collected in 1978 with 506 entries and 13 Variables. Each variable represents different features for homes in various suburbs in Boston, Massachusetts.


The purpose of this project was to first, compare the linear regression method to the random forest regression method to determine which is optimal. We discovered that the random forest regression is a better method because you can maintain higher accuracy with fewer data points than the linear aggression allows. Then, we created an app to allow users to evaluate the testing accuracy of the random forest regression using any number of factors from the dataset.


Here is a list of all the features in the original dataset:

  • Crime Rate: per capita crime rate by town
  • Zoning: proportion of residential land zoned for lots over 25,000 sq. ft.
  • Industry: proportion of non-retail business acres per town.
  • Charles River dummy variable (1 if tract bounds river; 0 otherwise)
  • Nitric Oxide: nitric oxides concentration (parts per 10 million)
  • Rooms: average number of rooms per dwelling
  • Age: proportion of owner-occupied units built prior to 1940
  • Distance: weighted distances to five Boston employment centers
  • Accessibility: index of accessibility to radial highways
  • Property tax: full-value property-tax rate per $10,000
  • Education: pupil-teacher ratio by town
  • Class: % lower status of the population
  • Median value of owner-occupied homes in $1000's
